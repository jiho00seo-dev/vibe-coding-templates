# 📘 AI Portfolio Vibe Pack: 초보자용 통합 매뉴얼

이 매뉴얼은 코딩을 전혀 모르는 분들이 **AI Portfolio**를 본인의 것으로 만들고, 웹사이트로 배포하여 수익화(쿠팡 파트너스)까지 완료하는 과정을 담고 있습니다.

---

## 🏗️ Step 1: 준비물
1. **구매한 패키지 파일**: `index.html`, `vibe-tokens.json`, `.env.example` 등
2. **Cursor 에디터**: [cursor.com](https://cursor.com)에서 다운로드 및 설치 (무료 버전 가능)
3. **Github 계정**: [github.com](https://github.com) 가입
4. **Cloudflare 계정**: [cloudflare.com](https://cloudflare.com) 가입

---

## 🎨 Step 2: 1분 만에 내 정보로 바꾸기 (개인화)

Cursor를 실행하고 패키지 폴더를 연 뒤, 우측 채팅창(Ctrl+L)에 다음 순서대로 입력하세요.

### 1. 기본 분석 지시
> "제공된 `index.html`과 `vibe-tokens.json`을 읽고 분석해줘. 나는 비개발자야."

### 2. 내 경력 정보 입력 (★중요)
아래 괄호 안의 내용을 본인의 정보로 바꿔서 입력하세요.
> "다음 정보로 사이트 내용을 전부 바꿔줘. 벤토 그리드 디자인이 예쁘게 유지되도록 문구 길이를 조절해줘.
> - 이름: [홍길동]
> - 역할: [데이터 사이언티스트 / AI 에이전트 개발자]
> - 핵심 슬로건: [내일의 지능을 오늘 설계합니다]
> - 주요 경력: [A사 챗봇 구축 프로젝트, AI 논문 3편 발표]
> - 강조 수치: [프로젝트 20개 완료, 정확도 95% 달성]"

### 3. 색상 및 분위기 변경
> "전체적인 Blue 테마를 [에메랄드 그린 / 오렌지 / 퍼플] 느낌으로 바꿔주고, `vibe-tokens.json`의 색상 값도 그에 맞춰서 업데이트해줘."

---

## 💰 Step 3: 수익화 연결 (쿠팡 파트너스)

가장 중요한 수익 창출 단계입니다.
1. 쿠팡 파트너스에서 원하는 상품(예: 맥북, 모니터)의 **링크**를 복사합니다.
2. Cursor 채팅창에 입력하세요:
> "Gear 섹션에 있는 장비들을 내 링크로 바꿔줘.
> 1. [장비이름]: [복사한 쿠팡 링크]
> 2. [장비이름]: [복사한 쿠팡 링크]
> 클릭하면 새 창에서 열리게 해주고, 클릭하고 싶게끔 버튼에 부드러운 애니메이션 효과를 더 넣어줘."

---

## 🌐 Step 4: 세상에 내 사이트 공개하기 (배포)

### 1. Github에 올리기
1. [Github.com](https://github.com) 로그인 후 **'New repository'** 클릭.
2. 이름(예: `my-ai-portfolio`) 입력 후 생성.
3. 생성된 페이지의 주소를 복사합니다.
4. Cursor 채팅창에 입력하세요:
> "방금 수정한 이 모든 파일들을 내 Github 레포지토리에 올려줘. 주소는 [복사한 Github 주소] 야."
> *(AI가 터미널 명령어를 알려주거나 대신 실행해 줄 것입니다.)*

### 2. Cloudflare Pages로 배포하기
1. [Cloudflare 대시보드](https://dash.cloudflare.com) 접속 -> **'Workers & Pages'** 클릭.
2. **'Create application'** -> **'Pages'** -> **'Connect to Git'** 클릭.
3. 방금 만든 Github 레포지토리(`my-ai-portfolio`) 선택.
4. **'Save and Deploy'** 버튼 클릭! (설정은 건드릴 필요 없습니다.)
5. 약 1분 뒤 나오는 `https://...pages.dev` 주소가 바로 당신의 웹사이트입니다.
